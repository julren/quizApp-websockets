# quizApp-websockets
Studienprojekt. WebApp für die Erstellung und Durchführung von Quizzes. Basierend auf *Angular (Frontend)*, *Node.js (Backend)*. Quizzes können zeitgleich mit mehreren Teilnehmern abgehalten werden. Dafür werden *Websockets* eingesetzt.

## Features:
* Registrieren/Login
* Rollenbasierter Funktionsumfang
* Dozenten können:
  * Quiz + Fragen anlegen/verwalten
  * Quiz aus vorhandenem Fragenpool zusammenstellen
  * Quiz sofort oder zu vorgegeben Zeitpunkt starten lassen
  * Auswertungen zu den Ergebnissen der Quizteilnehmer einsehen
* Quizteilnehmer (Studenten) können:
  * Sich mit Hochschulaccoutn einloggen (LDAP)
  * Freigeschatete Quizzes absolvieren
  * Ihre Ergebnisse einsehen
* Admins können:
  * User(rollen) verwalten
  * User als Admin/Dozent freischalten
